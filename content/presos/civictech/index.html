What UI framework to use?

How about a Government maintained OSS one?

  • Use US Web Design System as a base, build API wrapper
  • Started as consistent base, functionality, and styling for government websites
  • Open source, and a good base for gov-adjacent sites
  • Mostly front-end functionality with some batching and caching for API data
  • Asking regular users for feedback and incorporating those
