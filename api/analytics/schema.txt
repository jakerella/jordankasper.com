SITE ANALYTICS NETLIFY BLOBS DATA SCHEMA:

KEY: site-visitors
{
    "fingerprint": {
        g: String,       // geolocation (country code)
        u: String,       // userAgent details in the format: browser|os|format (i.e. "chrome|windows|desktop")
        f: Number,       // timestamp of first visit
        l: Number        // timestamp of latest visit
    },
    ...
}

KEY: site-paths
{
    "/path": String      // A unique ID used for each path
}

KEY(S): hits--CCYY-MM-DD
{
    "pathId": {          // unique paths visited on the website on this day
        h: Number,       // number of hits (page views, non-bot)
        b: Number,       // number of hits by a bot user agent
        v: [...String],  // fingerprints of unique, non-bot visitors
        q: [...String],  // query terms used for this path
        r: [...String]   // unique referral URLs
    },
    ...
}
